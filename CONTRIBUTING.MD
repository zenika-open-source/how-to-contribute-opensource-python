## Contributing to PyWeather App


Thank you for your interest in contributing to PyWeather App Here are some guidelines to help you get started.

## How to Contribute

1. Fork the repository.
2. Create a branch for your feature or bug fix (`git checkout -b my-feature`).
3. Make your changes and commit them (`git commit -m 'Add a new feature'`).
4. Push your branch (`git push origin my-feature`).
5. Open a Pull Request.

## Submission Rules

- Follow the project's code style.
- Add tests for any new features.
- Update the documentation if necessary.

## Reporting an Issue

To report a bug or request a new feature, please open an issue in the repository.
